====================================
How to generate report by envelopes?
====================================

.. toctree::

In this section you will know how to generate report by envelopes

If you want to get xlsx file with report byt envelopes you have to do the following:

1. Open platform

2. Got to mailbox page

3. Open tab with envelopes filter you need or filter data manually

4. Click on button with document icon (button called "Generate envelopes report")

5. After click you will see the message that report will be send after generation to you email address

6. Follow to email address

7. Open email you got

8. Click on button "Download"

Now you can open the file on your computer


File structure
==============

File structure contains two lists:

1. Filter details - there is you will see filter configuration you set on the web to make report

2. Report data - there is you will see envelopes list according to the filters above (similar data you see on Web)

Filter details structure
========================

1. Report date - date of the report generating (server time)

2. Status - the list of statuses which has been applied to the report

3. Period - period of time which has been applied to the report

4. Sender - the list of senders which has been applied to the report

5. Template - the list of templates which has been applied to the report

6. Subject - subject which has been applied to the report

7. Archive - status of the envelope archive or not which has been applied to the report

8. Labels - the list of labels which has been applied to the report

9. Expiration - period of expiration time which has been applied to the report

10. Scope - the scope on envelopes (inbox/outbox)

Report data structure
=====================

1. Subject - subject of the envelope

2. Template - template name of the envelope

3. Stage - stage on which envelope is in

4. Status - status of the envelope

5. Sender - name of the sender mailbox

6. Receive date - date of receiving an envelope
