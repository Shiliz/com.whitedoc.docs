Behaviour of data after deleting a user profile
===============================================

.. toctree::

**Delete User profile, Accounts and all related data**

After User profile deletion also will be deleted:

1. Accounts where user has role "Account owner".
2. Mailboxes.
3. Templates.
4. Dictionaries.
5. Contacts.
6. Aliases.
7. Envelope drafts.

Envelopes in process not be deleted.
